Project details

Rooting For Nature

Description

Rooting For Nature is a university project aimed at creating a website for the sale of drones specialized in pollination and tree planting. The project aims to support forest management, combat desertification and promote the benefits of pollination. Thanks to the use of advanced drones, an innovative solution is offered to improve biodiversity and promote the regeneration of ecosystems.

My role in the project

My colleagues and I took care of creating the design system, defining the visual and functional guidelines of the site. We developed layouts and mock-ups to outline the structure and visual appearance of the user interface. Furthermore, we took care of the code design of the site template, using HTML, CSS and Bootstrap to build a functional and responsive platform, which ensured fluid navigation and an optimal user experience.
