Project details

Webble

Description

Webble is a website born as part of a university project for a communications agency, designed with a playful and youthful theme. The goal of the site is to convey the energy and creativity of the agency, offering an engaging and dynamic user experience. Webble uses eye-catching graphics and an interactive interface to reflect the agency's innovative character and attract a young, creative audience.

My role in the project

This project represents my second coding job, where I was responsible for designing the entire design structure and creating mock-ups of the site. I implemented the code using Bootstrap for the first time, along with HTML, CSS and jQuery, to develop a responsive layout that adapts to different devices and screens. Furthermore, I collaborated on the creation of the contents, helping to define the tone and style, so that they were consistent with the young and fresh image of the agency. This project was an important opportunity to root my technical and creative skills in web design.
